Tipos de Seletores

Seletor de TAG

Ao usar o seletor de tipo TAG em H2 todos os elementos deste tipo vão herdar os estilos definidos pelo código CSS

Seletor do tipo .id

O seletor do tipo .id deve ser usado uma única vez no arquivo! Cada elemento pode ter apenas um ID. Cada página pode ter apenas um elementos com aquele ID

Seletor de Classe

Qualquer informação de estilo que precise ser aplicada a múltiplos elementos em uma página deve ser feita com uma classe. Você pode usar a mesma classe para vários elementos
